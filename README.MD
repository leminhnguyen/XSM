# XSM Project
Improving project forecast accuracy by integrating earned value
management with exponential smoothing and reference
class forecasting

![](xsm.jpg)

## Requirements
- Python 3
- numpy
- matplotlib
- pandas

## Installation
```
pip install -r requirements.txt
```

## Usage example
Run main.py file

```sh
python main.py C2013-12
```
